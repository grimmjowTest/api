🛠️ Tech Stack

- Java
- Spring Boot
- JPA (Hibernate)
- MySQL
- REST APIs

⚙️ Features

- CRUD operations for users, movies, and reviews
- Jakarta Validation (@Size, @NotNull, @Email)
- User–movie relationships (favorite movies)
- Input validation and global exception handling

🚧 Future Improvements

- JWT-based authentication and authorization
- Review DTO implementation for better data encapsulation
- Role-based access control (admin/user)
- Pagination and sorting for API endpoints
- Unit and integration testing
